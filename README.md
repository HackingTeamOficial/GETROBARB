🕷 GETROBARB XSS Scanner Framework v1.0

Framework automatizado de escaneo XSS para Hacking Team. Integra las mejores herramientas del ecosistema:

Workflow completo:

    📡 Subfinder → Enumeración de subdominios
    ✅ httpx → Verificación de hosts vivos (200, 301, 302, 403)
    🕸 Katana + Gospider → Crawling profundo de URLs
    📂 Waybackurls → URLs históricas desde Wayback Machine
    🔗 Merge → Unificación y deduplicación
    🔍 Params → Extracción de URLs con parámetros
    🧹 Urless → Limpieza de parámetros duplicados
    💉 Dalfox → Escaneo XSS automatizado

Uso:

./xss_scanner.sh all ejemplo.com    # Workflow completo
./xss_scanner.sh dalfox clean_params.txt  # Solo escaneo

Requiere: subfinder, httpx, katana, gospider, waybackurls, urless, dalfox
